生成SSH公钥，按照提示完成三次回车，即可生成 ssh key
ssh-keygen -t rsa -C "你的邮箱地址"
通过查看 ~/.ssh/id_rsa.pub 文件内容，获取到你的 public key
cat ~/.ssh/id_rsa.pub

git pull冲突
先将本地修改存储起来
git stash

查看本地仓库中都存储了几个stash版本
git stash list
还原暂存的内容
git stash pop stash@{0}
