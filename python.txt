--pipenv
-安装
pip3 install pipenv -i https://mirrors.aliyun.com/pypi/simple/

-修改 pipenv 虚拟环境安装位置
变量名：WORKON_HOME
变量值：D:\.virtualenvs

-cd 到项目文件夹目录下
-创建虚拟环境
pipenv install
或
pipenv --two
或
pipenv --three

-Pipfile 文件换源
url = "https://pypi.org/simple"
==>
url = "https://mirrors.aliyun.com/pypi/simple"

-激活虚拟环境
pipenv shell

-安装第三方库 并加入 Pipfile 的依赖清单
pipenv install xxx

-卸载第三方库 并清理 Pipfile 的依赖清单
pipenv uninstall xxx

-卸载全部包
pipenv uninstall --all

-显示当前依赖关系图
pipenv graph

-退出虚拟环境
exit

-删除虚拟环境，先 cd 进入 Pipfile 文件所在目录
pipenv --rm

--conda
-环境列表
conda env list

-当前环境的所有包
conda list

-创建虚拟环境
conda create -n xxx python=3.6

-删除虚拟环境及下属所有包
conda remove -n xxx --all

-激活默认环境
conda activate

-激活指定环境
conda activate xxx

-关闭当前环境
conda deactivate

-安装第三方包
conda install xxx

-卸载第三方包
conda remove xxx

-导出当前环境的包信息
conda env export > environment.yaml

-用配置文件创建新的虚拟环境
conda env create -f environment.yaml
